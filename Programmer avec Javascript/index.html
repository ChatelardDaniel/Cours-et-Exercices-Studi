Comprendre l’objet window

Les méthodes open() et close()

index.html

L’objet window permet d’ouvrir une fenêtre spécifique, mais aussi de fermer la fenêtre web actuelle.

Pour cela, nous pouvons utiliser les méthodes open() et close().


La méthode alert()

La méthode alert() est une méthode que vous connaissez déjà bien.

En général, nous l’utilisons sans le mot clé window, car comme nous l’avons dit, toutes les propriétés et méthodes de window peuvent être utilisées sans spécifier le mot clé, étant l’objet global par défaut.

Cependant, c’est une bonne pratique de le spécifier afin de bien mettre en avant que c’est une méthode de l’objet window.


La méthode prompt()
